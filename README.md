# AI Fake News Detector

## Project Overview
This project is aimed at detecting fake news using Artificial Intelligence techniques. The directory structure is organized to separate data handling, feature building, model training, evaluation, and app components to ensure clarity and maintainability.

## Directory Structure
- **data/**: Contains all datasets used for training and testing.
  - **raw/**: The raw, unpacked data.
  - **processed/**: The cleaned and processed datasets.
  - **external/**: External datasets.
- **notebooks/**: For Jupyter notebooks to visualize and analyze data.
- **src/**: Source code for the project.
- **app/**: Streamlit app files.
- **models/**: Serialized machine learning models.
- **reports/**: Figures and other report outputs.
- **tests/**: Test scripts for validating the project components.
